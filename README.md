# pc_on_stick
lets make a portable pc which will run on your pendrive, so you can use your personel os on almost any pc.


# Operating System
Operating system need hardware to work and a place to have its brain. Here hardware is like a physical body and we will use a small pendrive as a brain. A brain is a vessel to store out soul and here a pendrive will be a vessel to store our OS. pendrive will act as an actual hard-drive or SSD, so our pc can boot into it. we will install full os in pendrive so it will save our progress and keep it safe.


# STEP1- CHOOSE THE OS AND TOOL
We can use any os that can match your work or your vibe. Choose the os according to your pendrive speed and storage and needs. I have give the list in folder OS_types of os that can used for pc on stick with their speciafications and storage.

I am using MX linux because its all-rounder, easy to use and install. Download the **.iso** file from the link given in table.

The tool i am going to use is **rufus**, it is easy to use and fast tool. you can get its information and download lint in tools folder.

# STEP2- BOOT THE PENDRIVE
I this step, we are going to boot the pendrive with os. its like filling the brain with soul. We will first install the mx linux in pendrive and then do a full install.

Open rufus and set its parameters according to the image in image folder(do not select your internat ssd select pendrive with less storage) and click **start**.


# STEP3- FIRST BOOT
In this step, we will use **toram** method to free your usb.

Once the rufus says complete or done, follow these substeps.
step1- shut down your pc and start again.
step2- when your pc show the first screen or logo, press the **boot** key (F12, esc, F10). It will open the one-time boot menu.
step3- in boot menu go to boot options and select your pendrive, press enter.
step4- now it will take you to mx linux boot screen. here go to advanced option and look for **toram** option and enter it.

This process wil boot entire os in ram and lounch the os from ram. you can actuall remove usb without crashing system. Your pendrive is free now and the pc will boot in mx linux. however it will not store anything yet.
