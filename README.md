# pc_on_stick
lets make a portable pc which will run on your pendrive, so you can use your personel os on almost any pc.


# Operating System
Operating system need hardware to work and a place to have its brain. Here hardware is like a physical body and we will use a small pendrive as a brain. A brain is a vessel to store out soul and here a pendrive will be a vessel to store our OS. pendrive will act as an actual hard-drive or SSD, so our pc can boot into it. we will install full os in pendrive so it will save our progress and keep it safe.


# STEP1- CHOOSE THE OS
We can use any os that can match your work or your vibe. Choose the os according to your pendrive speed and storage and needs. I have give the list of os that can used for pc on stick with their speciafications and storage.

I am using MX linux because its all-rounder, easy to use and install.
